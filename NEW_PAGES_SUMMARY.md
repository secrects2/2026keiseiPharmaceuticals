# 新增頁面功能說明

## 📦 產品管理 (/admin/products)

### 功能概述
管理運動保健品的產品資訊與庫存

### 主要功能
1. **統計卡片**
   - 總產品數
   - 上架中產品數
   - 已下架產品數
   - 總價值統計

2. **產品列表**
   - 產品圖片展示
   - 產品名稱和編號
   - 分類和品牌
   - 單價和成本價
   - 上架狀態
   - 編輯和庫存管理按鈕

3. **搜尋和篩選**
   - 產品名稱搜尋
   - 分類篩選（營養補充品、運動器材、運動服飾）

### 資料來源
- `sports_products` 表

---

## 🤝 合作夥伴 (/admin/merchants)

### 功能概述
管理合作商家與合作關係

### 主要功能
1. **統計卡片**
   - 總合作夥伴數
   - 合作中數量
   - 洽談中數量
   - 平均佣金率

2. **合作夥伴列表**
   - 商家名稱和編號
   - 業務類型
   - 聯絡人資訊
   - 聯絡方式（電話、郵箱）
   - 佣金率
   - 合作開始日期
   - 合作狀態（合作中/洽談中/已終止）
   - 編輯和詳情按鈕

3. **搜尋和篩選**
   - 商家名稱搜尋
   - 狀態篩選

### 資料來源
- `partner_merchants` 表

---

## 💼 顧問服務 (/admin/consulting)

### 功能概述
管理健康諮詢與顧問服務活動

### 主要功能
1. **統計卡片**
   - 總服務數
   - 本月服務數
   - 即將到來的服務數
   - 已完成的服務數

2. **服務活動列表**
   - 活動名稱和描述
   - 活動日期和時間
   - 活動地點
   - 參與人數
   - 狀態（即將到來/進行中/已完成）
   - 編輯和詳情按鈕

3. **快速統計**
   - 近期活動列表（最新 5 筆）
   - 服務類型分布（運動諮詢、營養諮詢、健康講座）

4. **搜尋和篩選**
   - 活動名稱搜尋
   - 狀態篩選（即將到來/已完成）

### 資料來源
- `events` 表

---

## 🎨 設計特色

### 統一的視覺風格
- 使用 Tailwind CSS 設計
- 白色卡片 + 灰色背景
- Indigo 主色調
- 圓角和陰影效果

### 響應式設計
- 支援桌面和平板裝置
- 統計卡片使用 Grid 佈局（4 欄）
- 表格支援水平滾動

### 互動元素
- Hover 效果
- 狀態徽章（不同顏色）
- 操作按鈕（編輯、詳情、庫存）

## 📊 資料表結構

- contact_person, contact_phone, contact_email

- id, event_name, description
- event_date, location
- max_participants
- created_at, updated_at
